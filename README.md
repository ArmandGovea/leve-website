# Levé Beauty Bar — Booking Website

A modern, responsive beauty studio website with integrated appointment booking, real-time calendar management, and email notifications.

**Live:** [leve-beauty-bar.vercel.app](https://leve-beauty-bar.vercel.app)

---

## 🎨 Features

✅ **Responsive Design** — Mobile-first, works on all devices  
✅ **Interactive Calendar** — Real-time availability tracking  
✅ **Appointment Booking** — Multi-step form with validation  
✅ **Email Notifications** — EmailJS integration for booking confirmations  
✅ **Firestore Database** — Cloud-based booking storage  
✅ **Service Categories** — Brows, Nails, Lashes with multiple options  
✅ **Gallery** — Auto-scrolling image carousel  

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Database:** Firebase Firestore
- **Email Service:** EmailJS
- **Hosting:** Firebase Hosting / Vercel
- **Fonts:** Google Fonts (Cormorant Garamond, Jost)
